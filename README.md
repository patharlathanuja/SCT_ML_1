# House Price Prediction with Linear Regression

This repository contains a simple Python implementation of a linear regression model to predict house prices based on:

- square footage
- number of bedrooms
- number of bathrooms

## Files

- `linear_regression_house_prices.py`: trains a model, prints metrics, and predicts a sample house price.
- `requirements.txt`: dependencies for running the example.

## Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the script:
   ```bash
   python linear_regression_house_prices.py
   ```
