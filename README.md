## Project Title
House Price Prediction Using Linear Regression
## Description
This project focuses on developing a Linear Regression Model to predict house prices based on important housing features such as square footage, number of bedrooms, and number of bathrooms. The goal is to analyze the relationship between these features and property prices and use machine learning techniques to generate accurate price predictions.

The project involves data preprocessing, exploratory data analysis, feature selection, model training, and performance evaluation using various regression metrics. By leveraging the Linear Regression algorithm, the model learns patterns from historical housing data and predicts prices for new properties. This project demonstrates the practical application of supervised machine learning in the real estate domain and provides insights into factors that influence house prices.
## House Price Prediction with Linear Regression

This repository contains a simple Python implementation of a linear regression model to predict house prices based on:

- square footage
- number of bedrooms
- number of bathrooms
